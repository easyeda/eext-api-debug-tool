## 模块通信流程

```
模块 A                  事件总线                  模块 B
  │                        │                        │
  │  emit('data:updated') │                        │
  ├───────────────────────→│                        │
  │                        │                        │
  │                        │  callback(data)        │
  │                        ├───────────────────────→│
  │                        │                        │
  │                        │                        │  处理事件
  │                        │                        │
```

## 初始化顺序

```
1. 加载 core/ModuleSystem.js
   └─ 定义 WorkflowModuleSystem 和 WorkflowEventBus 类

2. 加载 core/init.js
   └─ 创建全局实例 window.workflowModuleRegistry

3. 加载所有 modules/*.js
   └─ 每个模块调用 register() 注册自己

4. 执行 bootstrap.js
   └─ 调用 initAll() 按依赖顺序初始化
      ├─ state (无依赖)
      ├─ canvas (无依赖)
      ├─ blockDefinitions (无依赖)
      ├─ render (依赖 canvas, state)
      ├─ interaction (依赖 canvas, state)
      ├─ toolbar (依赖 state)
      └─ example (依赖 state, canvas)
```
